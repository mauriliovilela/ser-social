## UC018 – Visualizar Foto ##

Este caso de uso é responsável por permitir o ator visualizar uma foto de um álbum.

**Ator:** Usuário.<br>

<b>Prioridade:</b> Importante<br>

<b>Requisitos Funcionais Associados:</b> <a href='RF007.md'>RF007</a> Gerenciar Fotos do Usuário.<br>
<b>Requisitos Não Funcionais Associados:</b> Disponibilidade, tempo de acesso, segurança do sistema, interface gráfica.<br>
<b>Pré-condição:</b> O ator deve estar conectado ao sistema (ter efetuado o login) e deve estar na página de álbuns.<br>
<b>Pós-condição:</b> O ator deve ter acesso a visualização de uma foto.<br>

<b>Fluxo de eventos principal</b><br>

1. O ator seleciona o álbum que contém a foto que deseja visualizar.<br>
2. O sistema retorna a página do álbum.<br>
3. O ator seleciona a foto que deseja visualizar.<br>
4. O sistema exibe a foto.<br>
5. O caso de uso é encerrado.<br>
